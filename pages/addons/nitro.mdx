# ND_Nitro
This resource creates a nos bottle that can be used on vehicles, when used the vehicle will be able to boost & purge.

## Setup guide
<Steps>
### Install the resource
Download and add the resource to your server
[ND_Nitro](https://github.com/ND-Framework/ND_Nitro/releases)



### Install dependency
* [ox_inventory](https://github.com/overextended/ox_inventory/releases)



### Create inventory items
```lua
["nos"] = {
    label = "NOS Bottle",
    weight = 2000,
    stack = false,
    close = true,
    client = {
        disable = { move = true, car = true, combat = true },
        usetime = 3500,
        cancel = true,
        export = "ND_Nitro.nos"
    }
},
```
