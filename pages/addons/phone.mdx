## Setup guide
<Steps>
### Install the resource
Download and add the resource to your server
[npwd](https://github.com/project-error/npwd/releases)



### Follow setup/configuration guide
[Project Error Documentation](https://projecterror.dev/docs/npwd/start/installation)



### Set framework
```copy filename="server.cfg"
set npwd:framework ndcore
```


### Set database
```json copy filename="config.json"
"database": {
    "useIdentifierPrefix": true,
    "playerTable": "nd_characters",
    "identifierColumn": "identifier",
    "identifierType": "license",
    "profileQueries": true,
    "phoneNumberColumn": "phonenumber"
},
```

</Steps>
