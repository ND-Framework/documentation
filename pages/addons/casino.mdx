## Setup guide
<Steps>
### Install the resource
Download and add the resource to your server
[ND_Casino](https://github.com/ND-Framework/ND_Casino/releases)


### Install dependencies
* [ox_target](https://github.com/overextended/ox_target/releases)
* [ox_inventory](https://github.com/overextended/ox_inventory/releases)
* [ND_AppearanceShops](https://github.com/ND-Framework/ND_AppearanceShops/releases)



### Create inventory items
```lua filename="items.lua" copy
["casino_chips"] = {
    label = "Chips"
}
```
