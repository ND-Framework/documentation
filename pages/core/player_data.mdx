# Player data

* id: `number`
* source: `number`
* identifier: `string`
* identifiers: `table`
    * steam: `player steam identifier`
    * license: `player fivem license`
    * xbl: `player xbl identifier`
    * discord: `player discord user id`
    * live: `player live identifier`
* discord: `table`
    * nickname: `string`
    * user: `string`
    * roles: `table`
* name: `string`
* firstname: `string`
* lastname: `string`
* fullname: `string`
* dob: `string`
* gender: `string`
* cash: `number`
* bank: `number`
* groups: `table`
* job: `string`
* jobInfo: `table`
* metadata: `table`
* inventory: `table`
